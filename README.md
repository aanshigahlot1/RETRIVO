## 🧭 What is RETRIVO?

> **RETRIVO** is a calm, privacy-aware web application designed to help people **report, discover, and recover lost items** in shared spaces — campuses, offices, and communities.

Unlike traditional lost-and-found boards, RETRIVO focuses on **verified recovery** — ensuring trust, transparency, and closure at every step.

---

## 😰 The Problem

In shared environments, losing something often leads to:

| ❌ Old Way | ✅ RETRIVO Way |
|---|---|
| Panicked WhatsApp/Telegram messages | Structured, centralized reports |
| Physical notice boards ignored | Live digital feed always accessible |
| No ownership verification | Email-confirmed recovery |
| Lingering unresolved listings | Auto-cleanup once recovered |
| Contact details exposed to everyone | Privacy-first — revealed only after verification |

## ⚙️ How It Works

```
  📝 Report Lost            📦 Report Found
       │                          │
       └──────────┬───────────────┘
                  ▼
          🔎 Smart Matching
          (name + location)
                  │
                  ▼
         📧 Email Sent to Owner
         (confirmation link)
                  │
                  ▼
         ✅ Verified Recovery
         Finder's contact revealed
                  │
                  ▼
         🗂️ Moved to History Archive
```

## 📂 Project Structure

```
RETRIVO/
├── 🏠 index.html       → Home page
├── 📝 lost.html        → Report lost items
├── 📦 found.html       → Report found items
├── 🔍 items.html       → Browse active lost & found listings
```

## 🔒 Privacy & Safety Design

```
🙈  Contact details hidden until ownership confirmed
🔐  Finder info shared ONLY after system verification  
🚫  No unnecessary data collection or exposed credentials
☁️   Serverless architecture reduces traditional security risks
```
